Intro

The media server is set up to be mostly automated, although a number of factors can prevent it from being fully automatic. I do my best to improve this, with the end goal being a server that requires no intervention on my part and minimal maintenance.

    Setup Guide

    The media server has a lot of different software and services running, so this section will provide you with some info on notable services and help you set them up. I may get a bit technical at times, but I do this so that you have a better understanding of how everything works.

    Tailscale

  • Tailscale is used to connect whatever device you are watching media on to the media server. The media server is connected, but not exposed to the internet, meaning the only way to access it is via Tailscale
  • Tailscale essentially makes a virtual network that connects devices. This means that even if we are across the world, our devices can see and talk to each other privately (no one else can see our computers talking)

    Setup

  • download Tailscale on your device using https://tailscale.com/download
  • after installing, create an account and then contact me so I can grant you access. When you do this, remind me to create you an account on Jellyfin as well, and I will respond with your username and password.

    Jellyfin

  • Jellyfin is a media streaming and management software that for all intents and purposes replaces Netflix. It gives the media server a familiar feel, making it easy to organize and stream movies/shows

    Setup

  • Download Jellyfin Client (Media Player) using the links below
    1. Windows: https://github.com/jellyfin/jellyfin-media-player/releases (you want "JellyfinMediaPlayer-1.9.1-windows-x64.exe")
    2. iOS: https://apps.apple.com/ca/app/swiftfin/id1604098728
    3. MacOS: https://github.com/jellyfin/jellyfin-media-player/releases (I have not tested this software on MacOS, but I believe it should work out of the box. You want "JellyfinMediaPlayer-1.9.1-macos-notarized.dmg")
  • Launch your Jellyfin Client (please ensure that Tailscale is both running and connected)
  • In Tailscale (https://login.tailscale.com/admin/machines) find "andyjellyfinserver" and copy the address (should be similar format to "100.xxx.xx.xx" (the x's will be numbers))
  • Enter the Tailscale address followed by :8096 in the host field (e.g. 100.xxx.xx.xx:8096) and click "Connect"

    2. :8096 is the port that Jellyfin runs on, meaning "100.126.93.52" is the address of the entire media server, and :8096 is the specific port that Jellyfin runs through
  • You should now be on the sign in screen. Enter the Jellyfin username and password I gave you earlier and click "Sign In"
    1. ![[Pasted image 20240214104130.png]]
    2. If you don't have an account, reach out to me and I can make you on (it takes like 30 seconds)
  • You should now be connected! Click on a movie or show, and test it out.

      Notes

  • Using the Jellyfin Client on your device helps to reduce the workload on the media server. Simply put, the client allows the server to send the media file directly/raw, as opposed to changing (transcoding) the file to a different format before sending it. Transcoding put strain on the media server, and using the client allows more people to use the server concurrently
